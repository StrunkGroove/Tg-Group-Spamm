Для запуска без лишней воды

1) Перейти на кладку: Accounts

2) Сменить у существующего бота текст и название сервера

3) Нажать кнопку start у бота

4) Ждать

PS: Рассылка обязательным пользователям должна прийти через несколько минут

Accounts

Страница на которой отображаются боты и информация о их прогрессе


Состоит: Формы и таблицы


Для добавления нового бота необходимо указать в форме и связаться со мной @StrunkGroove:

Phone - номер тлф для входа в учетную запись в Tg. Пример: +88888888888

Password - пароль от учетной записи. Пример: eUBRegbye834t


Status - имеет 2 состояния:

- Кнопка start. Имеет 2 условия для использования: необходимо указать имя сервера и текст для спама

- Дата начала парсинга группы

Server - отображает название группы.

Switch server - по нажатию кнопки всплывает модальное окно для ввода названия сервера. При смене текста удаляет всех пользователй и останавливает бота в течении 5 мин.

Switch text - по нажатию кнопки всплывает модальное окно для ввода текста для спама. При смене текста удаляет всех пользователй и останавливает бота в течении 5 мин.

Download - скачивает файл с пользователями

Delete bot - удаляет аккаунт

User take spamm - количество пользователей, которые получили спам-рассылку

Require Accounts

Страница на которой указываются пользователи, которые обязательно получат рассылку
